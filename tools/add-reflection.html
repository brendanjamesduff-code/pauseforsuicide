Pause Reflections JSON Tool

This tool helps you add a new reflection safely and quickly. Nothing is uploaded anywhere.

How to use
1) Copy your current reflections.json into the box below.
2) Paste the new reflection and click Generate + Insert at Top.
3) Click Copy Updated reflections.json and paste it back into GitHub.
4) Commit.
